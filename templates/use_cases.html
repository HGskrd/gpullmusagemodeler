Definition Versus Scale

A use-case kind defines workload dynamics: difficulty, SLO, price ceiling, capability gates, batchability, latent-demand economics, and token shape. The calculator keeps only the selected kind plus the organization's daily scale, so the same kind can be used at very different volumes.
